Session 22

Angular Pipes

Angular bisa melakukan formatting terhadap data yang dibungkus dengan interpolation, dengan menggunakan pipes.

Di sini, saya mencontohkan bagaimana cara melakukan piping, dengan pipe bawaan Angular (seperti DatePipe dan JSONPipe), maupun dengan custom pipe.

Buka source file ini di dalam:
src/app/pages/session22/session22.component.html

Pipe dengan Date

Date bawaan

{{ currentDate }}

Date dengan default DatePipe

{{ currentDate | date }}

Date dengan custom DatePipe

{{ currentDate | date: 'MMMM dd, YYYY' }}

Pipe dengan JSON

Tanpa JSONPipe

{{ person }}

Dengan JSONPipe

{{ person | json }}

Custom Piping

Person dengan PersonPipe

{{ person | personPipe }}
