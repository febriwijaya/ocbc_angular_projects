Angular Expressions

Angular dapat menjalankan expressions di dalam sebuah interpolation. Namun bukan berarti kita dapat menjalankan function ataupun operasi yang memiliki side-effect di dalam interpolation.

Perhatikan contoh di bawah ini!

{{ name | uppercase }}

{{ name | lowercase }}

Ternary inside interpolation

{{ name === 'Budi' ? name : 'name' }}

Tombol di bawah ini digunakan untuk mengganti nilai pada name, dan melihat ternary operator bekerja di dalam interpolation.
Ketika tombol ini diklik, akan mengganti nilai name menjadi Anto.
